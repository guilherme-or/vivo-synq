Arquitetura VIVO Atual

Como funciona?

O sistema segue um fluxo simples em que o cliente faz um pedido de dados no aplicativo da Vivo, a API busca os dados nos bancos de dados da empresa, e retorna as informações ao aplicativo se tudo for encontrado corretamente. Caso algum dado falte, nenhum dado é retornado.

Imagem do diagrama

Fluxo de Dados

  1. O cliente acessa o aplicativo da Vivo.
  2. O aplicativo envia uma solicitação à API.
  3. A API busca os dados nos bancos de dados.
  4. Se todos os dados forem encontrados, a API os transforma e retorna ao aplicativo.
  5. Se algum dado estiver faltando, a API não retorna nenhum dado.
  6. O cliente visualiza (ou não) as informações no aplicativo.

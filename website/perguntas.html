Q&A

Qual é o principal problema do sistema atual da Vivo que vocês estão resolvendo?
O sistema atual não retorna dados se algum banco falhar, deixando o cliente sem informações. Nosso cache resolverá isso.
O que é um sistema de cache e como ele melhora o desempenho?
O cache armazena temporariamente dados usados com frequência, acelerando o acesso e reduzindo a sobrecarga nos bancos de dados.
Como o sistema de cache resolverá o problema de "nenhum dado ou todos os dados" da API?
O cache fornece uma alternativa. Se algum banco falhar, os dados disponíveis no cache serão entregues ao cliente.
E se os dados armazenados no cache estiverem desatualizados?
O cache é atualizado regularmente, garantindo que os dados estejam sempre "frescos".
Quais benefícios o sistema de cache trará ao assinante da Vivo?
Mais velocidade, maior confiabilidade e uma experiência sem interrupções, já que o cliente não ficará sem informações.
